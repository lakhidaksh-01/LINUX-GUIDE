Introduction to Linux Operating System

  • Linux is a family of open-source, Unix-like operating systems that are based on the Linux kernel.
  • It was initially released by Linus Torvalds on September 17, 1991.
  • As a free and open-source operating system, its source code can be modified and distributed both commercially and noncommercially.
  • Originally created for personal computers, Linux has gradually been adopted for use in other machines such as servers, mainframe computers, and supercomputers.
  • Today, Linux is also utilized in embedded systems including routers, automation controls, televisions, digital video recorders, video game consoles, and smartwatches.
  • One of Linux's biggest successes is Android, an operating system based on the Linux kernel, which is widely used on smartphones and tablets.
  • Due to Android, Linux has the largest installed base of all general-purpose operating systems.
  • Linux is generally packaged in a form known as a Linux distribution.

Linux Commands You Should Know

  1. ls - The ls command lists directory contents.

    ls -l
    
# Common Options
-a: Lists all entries including hidden files (those starting with a dot '.').
-l: Uses a long listing format.
-h: When used with '-l', shows file sizes in human-readable format (e.g., KB, MB).
-R: Lists directories and their contents recursively.

  2. pwd - The pwd command prints the working directory.

    pwd

  3. cd - The cd command changes the current directory.

    cd dir_name

  4. mkdir - The mkdir command creates a new directory.

    mkdir new_directory

  5. rmdir - The rmdir command removes an empty directory.

    rmdir directory

  6. cp - The cp command copies files or directories.

    # It will copy a file called example.txt from the current directory to a directory called abc
    
cp example.txt abc/

  7. mv - The mv command moves or renames files or directories.

    mv old_name new_name

  8. touch - The touch command creates an empty file or updates the timestamp of an existing file.

    touch filename

  9. rm - The rm command removes files or directories.

    rm file

  10. clear - The clear command clears the terminal screen.

    clear

  11. ps - The ps command displays information about running processes.

    ps

  12. su - The su command allows you to switch user.

    su username

  13. sudo - The sudo command allows a permitted user to execute a command as the superuser or another user.

    sudo command

  14. vim - The vim command opens the Vim text editor.

    vim filename

  15. man - The man command displays the manual pages for Unix and Unix-like operating systems.

    man command_name

  16. cat - The cat command concatenates files and prints the output to the standard output.

    cat filename

  17. chmod - The chmod command changes the permissions of a file or directory.

    
# The first digit denotes permissions for the file/directory owner.
# The second digit denotes permissions for the group to which the file/directory belongs.
# The third digit denotes permissions for all other users.
# Permissions:
# 0 (no permission)
# 1 (execute only)
# 2 (write only)
# 3 (write and execute)
# 4 (read only)
# 5 (read and execute)
# 6 (read and write)
 # 7 (read, write, and execute)
                        
# Example:
# Change permissions of "file.txt" to 700 (read, write, and execute for the file owner, no permissions for group and others)
    
chmod 700 file.txt

  18. chown - The chown command changes the owner and group of a file or directory.

    chown new_owner filename

  19. tar - The tar command is used to compress and decompress files.

    
# x: Extract files from an archived package
# t: Display the contents of an archived package
# r: Add files to an existing archived package
# z: Apply gzip compression
# j: Apply bzip2 compression
# cf: Create an archived package with specified files
# xf: Extract specific files from an archived package
    
tar cf archive.tar file1 file2 file

  20. gzip - The gzip command compresses files using the gzip algorithm.

    gzip filename

  21. gunzip - The gunzip command is used to decompress files compressed by gzip.

    gunzip file.gz

  22. useradd - The useradd command is used to create a new user account.

    useradd username

  23. passwd - The passwd command is used to change a user's password.

    passwd username

  24. userdel - The userdel command is used to delete a user account.

    userdel username

  25. uptime - The uptime command shows how long the system has been running.

    uptime

  26. df - The df command displays disk space usage for filesystems.

    df -h

  27. du - The du command shows disk usage of files and directories.

    du -sh directory

  28. mount - The mount command mounts a filesystem.

    mount /dev/sdX /mnt

  29. unmount - The unmount command unmounts a filesystem.

    umount /mnt

  30. date - The date command displays the current date and time.

    date

  31. whoami - The whoami command prints the current user's username.

    whoami

  32. finger - The finger command displays information about user(s) on the system.

    finger username

  33. uname - The uname command prints system information.

    uname -a
    
# Common Options
-a, --ignore, provided for compatibility with other versions
-Z, --context, Display solely the security context of the current user                       
-g, --group , Output solely the effective group ID                      
-G, --groups, Output all group IDs          
-n, --name, Output a name instead of a number, used with -ugG         
-r, --real, Output the real ID rather than the effective ID, along with -ugG
-u, --user, Output only the effective user ID

  34. cat - The cat command concatenates files and prints the output to the console.

    cat filename

  35. history - The history command displays the command history of the current user.

    history

  36. echo - The echo command prints text or variables to the terminal.

    echo "Hello, world!"

  37. tee - The tee command reads from standard input and writes to standard output and files.

    command | tee file.txt

  38. locate - The locate command is used to find files by name quickly.

    locate filename

  39. uniq - The uniq command filters adjacent matching lines from input.

    uniq filename

  40. nl - The nl command numbers lines in a file.

    nl filename

  41. tr - The tr command translates or deletes characters.

    tr '[:lower:]' '[:upper:]'

  42. sed - The sed command is a stream editor for filtering and transforming text.

    sed 's/search/replace/g' filename

  43. cal - The cal command displays a calendar.

    cal

  44. cmp - The cmp command compares two files byte by byte.

    cmp file1 file2

  45. sort - The sort command sorts lines of text files.

    sort filename
    
# Common Options
-u, --sorted uniq values in ascending
-r, --sorted in decending                       
-unr, --shows only first characters                     
-k2, --no. of coloumns used in line        
-n, --sorted in ascending (default)

  46. wc - The wc command counts lines, words, and characters in a file.

    wc filename

  47. grep - The grep command searches for patterns in files.

    grep "search_pattern" file.txt

  48. head/tail - The head and tail commands display the beginning or end of files.

    head -n 10 filename
    tail -n 10 filename

  • All the commands execute sequentially.
  • Some task like file manipulation, program execution, user interaction, automation of task etc can be done

If-Else Statements

If-else statements are used to make decisions in your script.

# If-Else Example
if [ "$1" -gt 100 ]
then
echo "The number is greater than 100."
else
echo "The number is 100 or less."
fi


If-Else Example 2
if [ -e filename.txt ]
then
echo "file exist"
else
echo "file does not exist"
fi


Switch Cases

Switch cases (using `case` in shell scripting) allow you to execute different parts of code based on the value of a variable.

# Case Example
case "$1" in
    1)
        echo "Starting the service..."
        ;;
    2)
        echo "Stopping the service..."
        ;;
    3)
        echo "Restarting the service..."
        ;;
    *)
        echo "Usage: $0 {start|stop|restart}"
        exit 1
        ;;
esac


Loops

Loops are used to repeat a set of commands multiple times. There are three main types of loops in shell scripting: for loops, while loops, and until loops.

A for loop is used to iterate over a list of items.

# For Loop Example
for i in 1 2 3 4 5
do
echo "Iteration $i"
done

In this example, the loop iterates over the numbers 1 to 5, printing the iteration number each time.

# For Loop Example 2
for x in raj om shiv
do
echo "My name is $x"
done

In this example, the loop iterates over the words, printing the words each time.


While Loop

A while loop repeats as long as a specified condition is true.

# While Loop Example
count=0
num=10
while [ $count -le $num ]
do
echo "Numbers are $count"
let count++
done

In this example, the loop runs as long as $count is less than or equal to 10, incrementing the count each time.


Until Loop

An until loop repeats until a specified condition becomes true.

# Until Loop Example
count=0
num=10
while [ $num -eq $count ]
do
echo "$num"
let num--
done

In this example, the loop runs until $count is equal to 0, decrementing the count each time.


Infinite Loop

An infinite loop runs indefinitely and only stops when it is manually terminated. Here's an example using a while loop:

# Infinite Loop Example
while true
do
echo "Hi"
sleep 2s
done

In this example:

  • while true: The loop condition is always true, so the loop will run indefinitely.
  • do: This starts the body of the loop.
  • echo "Hi": This prints "Hi" to the terminal each time the loop runs.
  • sleep 2s: This pauses the loop for 2 seconds before the next iteration. The sleep command is used to delay the next command execution.
  • done: This ends the body of the loop.

You can stop an infinite loop manually by pressing Ctrl + C in the terminal where the script is running.



Functions

Functions allow you to group commands into reusable blocks. You can define a function using the following syntax:

function_name() {
# commands
}

To call a function, simply use its name. Here is an example:

# Function Example
greet() {
echo "Hello, $1!"
}

greet "Alice"

In this example, the function greet takes one argument and prints a greeting message. When called with greet "Alice", it outputs Hello, Alice!
